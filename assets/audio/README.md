# Audios de canciones

Coloca cada archivo como `.mp3` usando esta convencion:

`assets/audio/<saga-id>/<titulo-en-kebab>.mp3`

Ejemplos:

- `assets/audio/troy/the-horse-and-the-infant.mp3`
- `assets/audio/cyclops/polyphemus.mp3`
- `assets/audio/ocean/ruthlessness.mp3`
- `assets/audio/ithaca/i-cant-help-but-wonder.mp3`

La cancion `Would You Fall in Love with Me Again` queda reservada para un apartado especial y no debe ir en la lista de Ithaca.
